## ✨ Features that Define a New Benchmark

### 🛰️ Storage Central

Monitor your storage clusters with real-time holographic metrics. Precise, fast, and visually stunning.

Switch between a high-density **Tactical List** for precision operations and a premium **Aesthetic Grid** for media browsing.

Find your data nodes instantly with our high-speed filtering system.

Change the core accent colors of the entire system on-the-fly. Support for high-contrast OLED black themes and phosphorus neon palettes.

- **Storage Access**: SAF + Modern MediaStore Orchestration (API 10-14)
- **Animations**: Inifnite Transitions + GraphicsLayer 3D Rotations

## 🚀 Getting Started

### Installation

1. Download the latest **app-debug.apk** from the [GitHub Releases](https://github.com/jordelmir/ElysiumVanguard-FileManager/releases) section.
2. Install on any device running Android 10+ (API 29+).
3. Grant "All Files Access" when prompted for the full command-center experience.

### For Developers

```bash
git clone https://github.com/jordelmir/ElysiumVanguard-FileManager.git
cd ElysiumVanguard
./gradlew assembleDebug
```
