## Roadmap structure

The roadmap is organized by quarter.
For every quarter, the document
records:

- **The quarter.** (e.g. "2026 Q4")
- **The themes.** (e.g. "Phase 1
  foundational domain", "Phase 2
  vehicle definition", "Phase 4 AI
  council")
- **The outcomes.** (e.g. "G1 green",
  "G3 green", "G5 green")
- **The risks.** (per the risk
  register)
- **The retrospective.** (what
  shipped + what didn't + why)
